## API Agenda

### __Descrição:__
 * API REST em Java e MySQL que gerencia e disponibiliza uma agenda de contatos de pessoas.
### __Tecnologias Utilizadas__
 * Java
 * Spring Boot
 * Spring Data JPA com Hibernate
 * MySQL
 * Docker
